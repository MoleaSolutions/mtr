## Installation Notice
Because this application is not yet published or signed with Microsoft, Windows may display the following security warning during installation:

> "Windows protected your PC"
>
> "Microsoft Defender SmartScreen prevented an unrecognized app from starting. Running this app might put your PC at risk."

This warning is expected and does not indicate that the application is malicious.

To continue with the installation:

1. Click **More info**.
2. Click **Run anyway**.
3. Follow the installation prompts to complete the setup.

This warning appears because the application has not yet been digitally signed or published through Microsoft-recognized distribution channels.
